## User Interface

Autumn offers both **Internal** and **External** user interfaces to meet the needs of all types of users.

- The **Internal UI** is designed for quick, in-game access. Simply press **Insert** to open it up and start using the features right away.

- The **External UI** is for users who prefer to manage settings and scripts outside of the game.

No matter your style, Autumn has you covered.

# Autumn Custom Theming System

Welcome to the comprehensive guide for creating and using custom themes in Autumn! Our theming system allows you to completely transform the appearance of your Lua script editor with CSS.

## ðŸŽ¨ Getting Started

### Quick Setup

1. **Create the themes directory**: Create a folder named `theme` in the same directory as your Autumn executable
2. **Add your themes**: Place any `.css` file in the `theme` folder
3. **Automatic loading**: Themes are automatically detected and applied every second
4. **Live updates**: Modify your theme file and see changes applied instantly

## ðŸ› Troubleshooting

### Common Issues

1. **Theme not loading**: Ensure the CSS file is in the `theme` folder next to the executable
2. **Styles not applying**: Make sure all selectors start with `.custom-theme-active`
3. **Monaco editor styles**: Remember to use `!important` for Monaco editor overrides
4. **Performance issues**: Avoid excessive animations and complex selectors

### Debugging Tips

```css
/* Add debug borders to see element boundaries */
.custom-theme-active * {
  border: 1px solid red !important;
}

/* Test if your theme is loaded */
.custom-theme-active .editor-container {
  border: 5px solid lime !important;
}
```

## ðŸ“ Best Practices

1. **Theme Identification**: Always include a theme name comment at the top
2. **Consistent Naming**: Use descriptive CSS class names
3. **Performance**: Minimize complex animations and large background images
4. **Accessibility**: Ensure sufficient color contrast
5. **Testing**: Test your theme with different window sizes
6. **Fallbacks**: Provide fallback colors for gradients
